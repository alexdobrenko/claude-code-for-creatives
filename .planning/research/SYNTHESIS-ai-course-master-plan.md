## Frameworks Worth Teaching

### From Tina Huang
**TCREI Prompting Framework** — "Tiny Crabs Ride Enormous Iguanas"
- **T**ask — What you want AI to do
- **C**ontext — Background, images, details
- **R**eferences — Examples to inspire output
- **E**valuate — Review results
- **I**terate — Refine and improve

**Vibe Coding Framework** — "Tiny Ferrets Carry Dangerous Code"
- **T**hinking — Create PRD first
- **F**rameworks — Use established tools
- **C**heckpoints — Version control
- **D**ebugging — Screenshots + error messages
- **C**ontext — More = better

---

### From Greg Isenberg
**6-Step Startup Framework:**
1. Find idea (trend-based with Idea Browser)
2. Sketch it (TLDraw, 7 minutes)
3. Scope MVP (AI product manager like Manus)
4. Vibe code prototype (Bolt.new, 4 minutes)
5. Vibe market (AI SDR automation)
6. AI agent product manager (ongoing)

**Prompt Patterns:**
- "Ask me any questions before you get started" — Gets better output
- Image-enhanced prompting — Rough sketches make huge difference
- Persona-based consulting — "Pretend you're Warren Buffett"
- Context stacking — Upload courses, PDFs, documents

**The 80/20 Rule:**
> "AI does 80% of the work, you do the critical 20%. But you have so much time to make that last 20% much more impactful."
